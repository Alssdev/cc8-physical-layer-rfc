Welcome to **Physical Layer RFC**! This guide will help you set up the project locally and walk you through the basic steps to get started.

</Callout>

<Callout variant="warning">
  Make sure you set up `PICO_SDK_PATH` in your environment variables; otherwise, you will need to set it each time you compile.
</Callout>

## ğŸ› ï¸ Installation
Follow these steps to install and set up the project on your machine:

Clone your repository:

```bash
git clone https://github.com/Alssdev/raspberry-pi-pico-spi-daisy-chain.git
cd raspberry-pi-pico-spi-daisy-chain
```

Open the file `~/Makefile` and modify the next line accoarding your board.
```makefile
# set pico board
PICO_BOARD=pico_w       # pico_w or pico
```

Compile the code:

```bash
make
```

Now your code is ready to be loaded into your board! ğŸ‰

